## 🛠️ Tech Stack
- **Framework:** React Native (Expo SDK 50)
- **Navigation:** Expo Router
- **Language:** TypeScript

## 🚀 Cara Menjalankan
1. Clone repository ini.
2. Jalankan `npm install`.
3. Jalankan `npx expo start`.

## 📋 Fitur
1. Kartu Identitas Modern
2. Identitas Mahasiswa (Nama, NIM, Jurusan)
3. Animasi Fade-In
4. Verified Badge & Glow Effect
5. Personalized Quote & Emoji
6. Dark Theme Design
7. Responsive Layout
8. Digital Pulse Status
